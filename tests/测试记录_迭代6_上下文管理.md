## 测试1：`clean_tmp.py` 清理脚本

### 测试1a：正常清理

**步骤：**
1. 在 `output/tmp/` 下创建 3 个模拟临时文件
2. 先 `--dry-run` 预览
3. 再执行实际清理
4. 确认目录为空

**结果：**

```
=== 清理前 ===
  宗教效应_v1.md
  宗教效应_v2.md
  宗教效应_v3.md

=== --dry-run ===
  [dry-run] would delete: 宗教效应_v1.md
  [dry-run] would delete: 宗教效应_v2.md
  [dry-run] would delete: 宗教效应_v3.md
  3 files total, 3 would be deleted, 0 kept

=== 执行清理 ===
  deleted: 宗教效应_v1.md
  deleted: 宗教效应_v2.md
  deleted: 宗教效应_v3.md
  3 files total, 3 deleted, 0 kept

=== 清理后 ===
  ✅ 全部清理干净
```

**判定：** ✅ 通过

---

### 测试1b：`--days` 天数过滤

**步骤：**
1. 创建一个新文件 + 一个修改时间为 10 天前的旧文件
2. 执行 `clean_tmp.py --days 7`（只删 7 天前的）
3. 确认旧文件被删，新文件保留

**结果：**

```
=== 执行 --days 7 ===
  deleted: 旧文件_old.md
  2 files total, 1 deleted, 1 kept

✅ 10天前的旧文件已被清理
```

**判定：** ✅ 通过

## 测试2：长文本文件化流程模拟

模拟 Agent 收到用户文章后的完整流程：写入 → 读取 → 清理。

**步骤：**
1. Agent 收到用户文章（「艺术之争」总览，273 字）
2. Agent 用 `write_file` 写入 `output/tmp/艺术之争_v1.md`
3. Agent 需要回顾时用 `read_file` 读取
4. 流程结束后 `clean_tmp.py` 清理

**结果：**

```
文章长度：273 字
写入临时文件：...\output\tmp\艺术之争_v1.md
读取成功：273 字
✅ 全部临时文件已清理
```

**判定：** ✅ 通过

**模拟步骤：**
1. 连续 6 轮模拟（初稿 → 反馈 → v2 → 反馈 → v3 → 反馈 → v4 → 反馈 → v5 → 反馈 → v6 → 反馈）
