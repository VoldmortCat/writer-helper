# AI 写作搭子 — Hermes Agent Skill

选题 #21 — **AI 写作搭子**（来自 AI 实践场景清单）

## 项目简介

AI 写作搭子是一个**编辑伙伴 + 灵感催化剂**，封装为 Hermes Agent Skill。
你写一段，AI 给反馈；你卡壳了，AI 帮你梳理思路；你写长篇了，AI 帮你检查前后矛盾；
你写久了，AI 学会你的风格。

**AI 核心价值：** 不是套个 AI 壳做传统功能——AI 本身就是分析引擎。反馈的质量（识别结构问题、判断人物是否立住、发现论证漏洞）、风格的把握、灵感的个性化方向，这些没有 AI 根本做不到。

## 运行环境

| 项目 | 说明 |
|------|------|
| 运行平台 | Hermes Agent（CLI / 桌面应用 / 网关） |
| 依赖模型 | 任意具备 tool-calling 能力的 LLM（推荐 DeepSeek，学生有免费额度） |
| Python 版本 | 3.11.5（仅 scripts/stats.py 需要，核心功能无需 Python） |
| 额外依赖 | 无（无需 pip install、无需额外 API Key 配置） |

> **为什么不需要 API Key？** 本 Skill 走的是 Agent 指令路线——分析能力由 LLM 在 prompt 中直接完成，不需要中间脚本层调外部 API。Agent 本身已经配置了模型，Skill 只需要告诉 Agent 怎么写分析，不需要自己调模型。

## 核心流程

| 流程 | 触发短语 | 说明 |
|------|---------|------|
| **快速审稿** | "帮我看看这段" / 直接贴文字 | 5 维度分析 + 标出最需要改的一个问题 |
| **打磨跟单** | "帮我改改" | review -> 你改 -> 对比 -> 再改，直到你说停 |
| **卡壳疏导** | "写不下去了" / "没思路" | 给 3 个发展方向 + 示例开头 |
| **长篇体检** | "帮我检查整篇" | 多章一致性检查：人物矛盾/情节漏洞/伏笔回收/细节照应 |

## 使用方式

将本 skill 目录放入 Hermes Agent 的 skills 目录，然后在对话中：

```
你：帮我看看这段（贴文章内容）
Agent：5 维度分析 + 指出最需改的问题
→ 问：要按这个方向改一版吗？

你：（贴修改版）
Agent：对比原版和修改版，逐项评估改动效果
→ 问：还要继续调吗？
```

具体使用说明详见 `references/使用指南.md`。

## 迭代历程

| 迭代 | 方向 | 来源 | 核心变化 |
|------|------|------|---------|
| 1 | 文本类型自动检测 | 自发现 | review 适配不同文体（叙事/议论/混合） |
| 2 | 一致性检查 | xlsx 迭代1 | 长篇体检：人物矛盾/情节漏洞/伏笔回收 |
| 3 | 风格学习 | xlsx 迭代2 | 风格存档 + 新文章一致性检查 |
| 4 | 个性化灵感 | xlsx 迭代3 | 基于用户历史写作和风格给方向 |
| 5 | 流程化跟单 | 自发现 | 从功能罗列到 4 个跟单流程 + 自动切换 |

详见 `iteration/iteration_log.md`。
