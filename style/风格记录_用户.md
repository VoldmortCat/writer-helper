## 一、核心风格定性

**直接、犀利、不装逼的思辨型写作者。**
用大白话说深道理，敢用狠词，不怕得罪人。所有文字都透着一个态度：
"别跟我整那些虚的"。

---

## 二、用词特征

### 标志性高频词

| 词 | 出现频次 | 用法说明 |
|----|---------|---------|
| "大拿" | 高频 | 指代权威人物，带讽刺或谨慎的尊重 |
| "谜语人" | 自创词 | 指故意把话说得玄乎的人 |
| "弱智" | 多处 | 对某种行为或人的直接评价 |
| "傻逼" | 多处 | 同上的更强版本 |
| "装逼" | 多处 | 核心批判对象 |
| "我私以为" | 标志性 | 表达个人观点时的引导词 |
| "牛逼" / "贼吊" | 口语 | 褒义时使用 |

### 自创/活用表达
你是那种**不介意自己造词**的写作者：
- "越粉越深" — 把"粉"当动词用
- "粉它的程度" — 同上
- "物" — 加引号作为特定概念使用
- "隔层纱后面有东西" — 形象化的口语表达

## 三、句式特征

### 长短句结合，偏爱长句

典型长句模式：
```
一个长句开头 + 逗号连接多个子句 + 一口气说完 + 括号补充
```

示例：
> "一幅名家的作品，在不同人眼里都是不一样的，因为每个人不同的性格不同的经历会致使每个人对于一副画面都有着不同的感受，画家画面中所显示出来画家自身所认为的坚强，在不同人眼中有不同的思考理解"

读到一半要换气，但逻辑是完整的。

### 句式节奏

- **开头常用短句定调**："艺术这个词不应该被高级化"、"很反感"
- **论证时自动转入长句**，信息密度骤增
- **收尾时常用总结句或反问**

### 括号使用（核心特征）

括号是你的标志性工具，用途包括：
1. **随时补充背景**："(每个人对于一个词的定义是不同的)"
2. **自我纠正**："(注意哦，是深邃布局，不是深邃内容，他又不是哲学家)"
3. **举例说明**："(鱼桑混合，是叫这个吧，反正是一个生态循环可持续的生产模式)"
4. **拓展注释**："[拓展一]" 作为独立注释段落

## 四、语气和态度

### 总体：直接、情绪化但不失控

| 特征 | 表现 |
|------|------|
| 直白程度 | 高 — 不铺垫，不绕弯，上来就亮观点 |
| 情绪色彩 | 强 — "恶心""弱智""傻逼"自然出现，但用于论证不是纯骂街 |
| 自我定位 | 清醒的 outsider — "我私以为""我觉得"，不假装客观 |
| 对读者态度 | 平等的 — 不端着，不试图说服你，而是"我说你听，你自己判断" |

### 标志性语气词
- "纯纯是个……": 带讽刺的判断句
- "搁哪……呢": 口语化反问
- "狠狠提供给你": 带画面感的描述

## 五、修辞习惯

### 常用手法

1. **具体类比**（最突出）
   - "就像你家人拿你的成就和别人吹牛一样"
   - "马桶上印一个supreme，潘家园的核桃一样卖高价"
   - "一坨屎加了这层包装也能变成艺术"

2. **排比/列举**
   - "可以是坚强，也可以矫揉造作，也可以说是懦弱"
   - "文学、语言、绘画、舞蹈、音乐、雕塑……"

3. **设问/自问自答**
   - 问一个问题然后自己回答，推动论证前进

### 少用手法
- 少用比喻（但类比用得多）
- 少用典故/引用（偶尔提名人或作品，但不掉书袋）
- 很少用反问句结尾（你倾向于直接陈述，不留给读者悬念）

## 六、结构和组织

### 典型段落结构

```
亮观点（短句）→ 展开论证（长句）→ 举例/类比（具体）→ 括号补充（可选）→ 总结/下结论
```

示例（宗教效应）：
> "我脑中不由地总结出一个很泛的词——'宗教效应'"（亮观点）
> → "实质:通过打造出一个鹤立鸡群的'物'……"（展开论证）
> → "就像你家人拿你的成就和别人吹牛一样"（类比）
> → （形成了圈子……归属感）（补充总结）

### 文章结构

- **偏爱"现象→概念→分析"的递进结构**
- 开头总是从具体场景或观察切入，不是抽象定义
- 结尾通常有一个"所以"式的总结，不是悬念式的留白
- 喜欢附上写作时间和场景（"以上截止到2024.2.18 3:11"）

---

## 七、标点习惯

| 标点 | 使用特点 |
|------|---------|
| 引号 "" | 高频，用于标记"特定概念"或反讽 |
| 括号 () | 极高频，补充/解释/自我纠正 |
| 破折号 —— | 中频，用于引出解释或总结 |
| 逗号 ， | 超高频，长句内连接子句 |
| 句号 。 | 相对少用，段落内常以逗号一气呵成 |

---

## 八、一句话总结

**一个会说人话的思辨者——观点锋利、论证具体、不装、敢骂，但骂得有逻辑。**

---

## 九、写作建议（从你的风格出发）

如果你是 AI，要模仿这个风格写一段话，规则是：
1. 开头直接，不用铺垫
2. 过程中用括号随时补充
3. 举一个具体的、生活化的例子
4. 不怕用口语词（傻逼/装逼/弱智）
5. 但必须有用逻辑支撑，不能纯情绪输出
6. 结尾总结到位，不拖泥带水
7. 保持"我说我的，你爱信不信"的态度
