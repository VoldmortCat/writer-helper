# AI 写作搭子 — 使用指南

## 简介

AI 写作搭子是 Hermes Agent 的文字指引技能（Skill）。
它不是代写工具——你写，AI 给反馈；你卡壳了，AI 帮你找方向。

所有分析能力由 Agent 的 LLM 直接完成，无需额外安装 Python 依赖或配置 API Key。

## 使用方式

在 Hermes Agent 的对话中直接贴入你的文字，Agent 会根据你的意图自动选择流程：

### 1. 快速审稿

贴一段写好的文字，然后说"帮我看看这段"。

Agent 会自动判断文本类型（议论文/叙事文/混合），
从 5 个维度给出反馈，并标出最需要改的一个问题。

**适合：** 写完一段想听听意见，不需要反复改。

### 2. 打磨跟单

说"帮我改改"，Agent 会进入打磨模式：

1. 首次出 5 维度反馈 + 标出最需改的问题
2. → 问"要按这个方向改一版吗？"
3. 你改完后贴回来，Agent 逐项对比原版和修改版
4. → 问"还要继续调吗？还是这版可以了？"
5. 循环直到你说停

**适合：** 想反复打磨到满意。

### 3. 卡壳疏导

说"写不下去了"或"给点方向"，Agent 会：

1. 读取你当前写的内容
2. 查看你的风格记录（如果有的话）
3. 给 3 个发展方向 + 每个方向的示例开头
4. → 问"选哪个方向？"
5. 你选了方向后接着写，写完自然切到打磨跟单

**适合：** 卡壳了、没思路了。

### 4. 长篇体检

写完多章后说"帮我检查整篇"，Agent 会：

1. 读取你提供的所有章节
2. 检查 4 类问题：人物设定矛盾、情节逻辑漏洞、伏笔回收、细节照应
3. 输出检查报告，标注严重程度
4. → 问"要针对这些问题修改吗？"

**适合：** 写了长文/多章，担心前后矛盾。

## 风格记录

如果想长期使用，可以给 Agent 几篇你的旧文，
说"学一下我的写作风格"。Agent 会把你的风格特征
记录到 style/风格记录_用户.md 中，后续写新文章时
Agent 会自动参考，保持风格一致。

## 核心原则

| 原则 | 说明 |
|------|------|
| AI 给建议，你做决定 | 反馈是参考，最终改不改你来定 |
| 不是代写 | 本工具帮你想清楚，不是替你写 |
| 流程是导航不是镣铐 | 想看反馈就看反馈，想改就跟着改，说停就停 |
