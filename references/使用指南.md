# AI 写作搭子 — 使用指南

## 简介

AI 写作搭子是一个命令行工具，用 AI 帮你改进写作。它不做代笔——你写，它给反馈，卡壳时帮你找方向。

## 安装

### 依赖

```bash
pip install requests
```

### 配置 API Key

推荐使用 DeepSeek（免费额度够用）：

```bash
# Windows CMD
set WRITING_BUDDY_API_KEY=sk-你的key

# Windows PowerShell
$env:WRITING_BUDDY_API_KEY="sk-你的key"
```

> 如何获取 DeepSeek API Key？
> 1. 访问 https://platform.deepseek.com/
> 2. 注册 → 实名认证获得免费额度
> 3. 创建 API Key

## 使用

### 1. 反馈模式

写完一段后跑这个命令：

```bash
cd skill/scripts
python writing_buddy.py review --file 我的文章.txt
```

或者直接传文字：

```bash
python writing_buddy.py review --text "这是一个测试段落。我想知道AI能不能发现我的故事节奏有点慢。"
```

输出示例：

```
=== 结构 ===
开头抓住了注意力，但第二段到第三段的过渡有点生硬。

=== 人物 ===
主角的对话语气前后不一致，前面很暴躁后面突然很冷静。

=== 细节 ===
环境描写基本没有，读者不知道这场对话发生在哪里。

=== 节奏 ===
前两段推进很快，但第三段开始啰嗦，一句话绕了三个弯。

=== 语言 ===
"但是"用了四次，换几个词比如"然而""不过"会更流畅。

=== 最需要改的一个问题 ===
第三段啰嗦——砍掉一半字数，读者会感谢你。
```

### 2. 灵感模式

卡住时用这个：

```bash
python writing_buddy.py brainstorm --context "我写到主角发现死者是自己失散多年的父亲，现在他不知道该报警还是自己查。"
```

输出示例：

```
你的核心问题：主角在"法律义务"和"个人情感"之间的冲突。

方向一：报警但暗中查
  主角选择报警，但私下保留了关键证据。
  效果：张力在线、符合人物道德设定
  风险：容易写成俗套警察戏

方向二：先查再决定
  主角独自调查，想搞清楚父亲这些年经历了什么再说。
  效果：悬疑感强、能展开故事
  风险：需要合理理由解释为什么不报警

方向三：有人替他选了
  第三者（比如母亲/姐姐）得知后抢着报了警，主角被迫卷入。
  效果：外部冲突+内部冲突双重压迫
  风险：可能削弱主角主动性
```

## 常见问题

**Q: 没有 API Key 能试用吗？**
A: 不能，必须有 API Key 才能调用 AI。

**Q: 支持其他模型吗？**
A: 支持，设置环境变量：
```bash
set WRITING_BUDDY_API_BASE=https://dashscope.aliyuncs.com/compatible-mode/v1
set WRITING_BUDDY_MODEL=qwen-turbo
```

**Q: 能检查英文写作吗？**
A: 目前 system prompt 是中文评中文。改 system prompt 就可以评英文。

**Q: 我的文章很长怎么办？**
A: 目前每次最多处理 4000 字左右（8000 token）。超长文章建议分段检查。
